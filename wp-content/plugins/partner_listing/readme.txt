Featured Post Plugin for Wordpress.

== Description ==
Display partner name in post type by calling custom field partner name value in listing in Admin section. Just you need to add post type key at line no. 73 in array.

*Supports Custom Post Type*

Visit us on [GitHub](https://github.com/ssovit/featured-post) to support and donate 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
